# Sentiment Analysis AI

Sentiment Analysis AI is a natural language processing project that classifies text sentiment as positive or negative.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- NLTK

## Project Structure

- data/ - dataset files
- notebooks/ - exploration and preprocessing scripts
- models/ - saved trained models
- app.py - user interface
- sentiment_model.py - prediction logic
- train.py - training pipeline
